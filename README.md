# asdf-win

## Requirements:

- PowerShell 7.3.x
- Git

```shell
winget install Microsoft.PowerShell
```
```shell
winget install Git.Git
```

## Install
`Use PowerShell 7.3.x`
```shell
git clone https://github.com/ordnaelmedeiros/asdf-win.git "${HOME}/.asdf" && ."${HOME}/.asdf/install.ps1"
```
Restart PowerShell
